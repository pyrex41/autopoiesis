## 1. PR #12 — Content-Addressed Sandbox + Command Center Redesign

Reviewed and fixed 6 issues in PR #12 (+3,692 lines):

- **Format string bug**: `~~D` → `~D` in `diff-snapshots` (literal tilde instead of format directive)
- **Broken restore endpoint**: `rest-sandbox-restore` ignored the `snapshot-ref` URL parameter
- **Docker error handling**: `docker create` silently ignored failures
- **Unused import**: `onCleanup` in CommandView.tsx
- **Duplicated utility**: Extracted `formatSize` to shared `lib/format.ts`
- **Stale holodeck reference**: `AgentDetail.tsx` referenced removed `ViewId`

Also found and fixed a post-merge compilation issue: `sandbox-tools.lisp` and `sandbox-routes.lisp` used `autopoiesis.sandbox:` package-qualified symbols at read time, failing when the sandbox system wasn't loaded. Converted to `find-symbol` dynamic resolution.

All uses dynamic resolution (`find-package`/`find-symbol`) so the eval system loads without sandbox backends.
