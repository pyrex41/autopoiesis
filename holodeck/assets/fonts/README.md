# Holodeck Fonts

Download these fonts (all SIL Open Font License) from Google Fonts:

1. **Orbitron** (headers/titles) - `Orbitron-Regular.ttf`, `Orbitron-Bold.ttf`
   https://fonts.google.com/specimen/Orbitron

2. **Share Tech Mono** (labels/status) - `ShareTechMono-Regular.ttf`
   https://fonts.google.com/specimen/Share+Tech+Mono

3. **JetBrains Mono** (code/data) - `JetBrainsMono-Regular.ttf`
   https://fonts.google.com/specimen/JetBrains+Mono

4. **Rajdhani** (HUD numbers) - `Rajdhani-Regular.ttf`, `Rajdhani-Bold.ttf`
   https://fonts.google.com/specimen/Rajdhani

Place .ttf files directly in this directory. The egui theme will load them when available,
falling back to Bevy's default font otherwise.
